### Features

- **Daily task tracking** with projects, tags, and priorities
- **Time tracking** per task with start/stop timers
- **Mood & energy logging** to spot patterns
- **Learnings journal** tied to your daily work
- **XP & achievements** system to keep you motivated
- **Dashboard** with charts and productivity insights
- **Data export/import** for full backup and restore
- **Works offline** — all data stored locally in your browser

### Tech Stack

React, Vite, Tailwind CSS, Dexie (IndexedDB), Framer Motion, Recharts

### Getting Started

```bash
npm install
npm run dev
```

### Data Safety

Your data lives in your browser's IndexedDB. Go to **Settings** to export a JSON backup anytime. The app will remind you if it's been a while since your last backup.
